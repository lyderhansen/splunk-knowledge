**Add new function makeGradientRows(w, h, topLeftRgb, bottomRightRgb, accentRgb, accentCx, accentCy, accentR):**
Place it after the existing makeRgbRows function (after line 144). This function generates a diagonal gradient from topLeftRgb to bottomRightRgb with a radial accent glow centered at (accentCx, accentCy) with radius accentR.

Algorithm (from RESEARCH.md):
- For each pixel (x, y): compute ty = y / (h - 1), tx = x / (w - 1), t = (tx + ty) / 2
- Linear interpolation: r = Math.round(topLeftRgb[0] + (bottomRightRgb[0] - topLeftRgb[0]) * t) for each channel
- Radial accent overlay: compute dist = sqrt((x - accentCx)^2 + (y - accentCy)^2); if dist < accentR, apply glow = (1 - dist/accentR)^2 * 0.08 blend toward accentRgb
- Return rows in same format as makeRgbRows (array of h arrays, each w*3 bytes)

**Add new function generateGradientBg(appDir, dark):**
Place it after the existing generatePreviews function (after line 523). This function:
1. Computes gradient parameters from the dark theme:
   - topLeftRgb = hexToRgb(dark.bg) -- brand background as gradient start
   - For bottomRightRgb: blend dark.bg toward dark.panel at 60% (or use dark.panel directly if available, falling back to a 2-stop-brighter version of dark.bg). Compute as: Math.round(bgRgb[i] + (panelRgb[i] - bgRgb[i]) * 0.6) for each channel. If dark.panel is not available, use dark.bg directly (no gradient, just solid -- fallback).
   - accentRgb = hexToRgb(dark.accent) -- brand accent for radial glow
   - accentCx = Math.round(1920 * 0.3) -- glow center at 30% from left (hero zone area)
   - accentCy = Math.round(1080 * 0.2) -- glow center at 20% from top
   - accentR = 600 -- large radius for subtle ambient glow
2. Calls makeGradientRows(1920, 1080, topLeftRgb, bottomRightRgb, accentRgb, accentCx, accentCy, accentR)
3. Calls makePng(1920, 1080, rows) to encode the gradient as PNG
4. Creates output directory: path.join(appDir, 'appserver', 'static', 'images'). Use fs.mkdirSync with { recursive: true }.
5. Writes the PNG buffer to path.join(appDir, 'appserver', 'static', 'images', 'bg_gradient.png')
6. Logs: process.stdout.write('  wrote: ' + outPath + '\n')
