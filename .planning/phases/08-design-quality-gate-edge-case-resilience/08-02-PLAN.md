From 08-RESEARCH.md — Code Examples section:

ECR-01 drawEmptyState signature:
  drawEmptyState(ctx, w, h, t, accent)
  - dashed circle: cx=w/2, cy=h*0.38, r=Math.min(w,h)*0.1
  - ctx.strokeStyle = theme.withAlpha(accent, 0.20), lineWidth=1.5, setLineDash([4,4])
  - text: fontSize=Math.max(11, Math.min(16, h*0.07)), font uses theme.FONTS.ui
  - fillStyle = theme.withAlpha(t.textFaint, 0.7), textAlign='center'
  - text content = 'No data available'
  - Call site in updateView: check if !data and _lastGoodData is null, then call drawEmptyState and return

ECR-03 pagination:
  - State: this._currentPage = 0 in initialize()
  - maxRows = parseInt(opt('maxRows', '20'), 10); guard isNaN or < 1 → default 20
  - totalPages = Math.max(1, Math.ceil(totalRows / maxRows))  ← Pitfall 6 guard
  - Clamp: if this._currentPage >= totalPages → this._currentPage = 0
  - Slice: startIdx = this._currentPage * maxRows; endIdx = Math.min(startIdx + maxRows, totalRows)
  - pageRows = data.rows.slice(startIdx, endIdx)
  - Render only pageRows in the draw loop
  - Controls: draw prev/next only when totalPages > 1

ECR-04 safeStr/safeNum:
  - table cell: safeStr(row[colIdx[fieldName]])
  - numeric: safeNum(row[colIdx[fieldName]], 0)
  - WRONG: row[colIdx[field]] directly

ECR-05 ctx.save/restore:
  - Correct: ctx.save() before shadow-setting; ctx.restore() after draw
  - Correct: ctx.save() before globalAlpha change; ctx.restore() after
  - Wrong: manual reset with ctx.shadowBlur = 0 (misses cases when code paths diverge)
