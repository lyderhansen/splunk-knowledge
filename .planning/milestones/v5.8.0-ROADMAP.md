## Key Accomplishments

1. **JSONata expression reference (Phase 38)** — Built `ds-ref-jsonata` SKILL with full syntax: operators, string/numeric/date/array functions, lambdas, path expressions, plus 6 named recipes (RAG thresholds, dynamic labels, toggles, time arithmetic, conditional formatting, multi-token aggregation). `ds-int-tokens` MUST-LOAD directive wired so any `eval`/`conditions` work routes through the canonical reference. Closes the "JSONata is NOT SPL eval" trap.

2. **Extension API templates corrected (Phase 39)** — Three live-tested findings from v5.7.0 landed in templates: IIFE format (not ESM), bare stanza names in `visualizations.conf`, bundled `@splunk/dashboard-studio-extension` (not external). Every new Extension API viz now scaffolds with the correct format.

3. **Animation Helper Scope Rule (Phase 40)** — Documented AF-01/AF-02 in `animation-recipes.md` with WRONG/RIGHT framing: `opt()` only available inside `updateView`; helpers MUST receive computed values as parameters. Refactored all 5 animation helper signatures. Eliminates the silent `ReferenceError: opt is not defined` failure mode that hit every v5.5-5.6 animated pack.

4. **Pillow-based preview pipeline (Phase 41)** — New `generate_previews.py` writes per-viz 116×76 RGB miniature renderings using bundled Inter Regular 4.0 TTF (407 KB) — gauges, bars, lines, tables, KPI tiles, rings, generic. 3-tier detection cascade: declared `@viz-type` → keyword match → generic fallback. `generate_assets.js` preview path gated behind `--legacy-previews` flag. Plugin bumped 5.8.0 → 5.9.0.

5. **THM-05 light-mode `backgroundColor` fix (Phase 42)** — Generated viz code now reads `opt('backgroundColor')` BEFORE the theme branch, so both dark and light modes respect user-set background. Landed across 4 template files (theme-template, pre-code-checklist, visualization-js-template, config-json-template) with cross-link footers. Plugin bumped 5.9.0 → 5.9.1.

6. **Deep Review milestone audit (Phase 43)** — Exhaustive file-by-file review of every plugin file touched by v5.5-v5.8: 95 files reviewed across 6 waves (vp-viz references, vp-* SKILLs, vp-viz scripts, vp-create scripts, dashboard-studio bounded set, tests sweep). Produced `43-REVIEW.md` with **19 BLOCKERs + 30 WARNINGs + 24 NITs**, every BLOCKER+WARNING fixed inline across remediation plans 43-07 through 43-12. Plugin bumped 5.9.1 → **5.10.0** (splunk-viz-packs) + 3.4.0 → **3.5.0** (splunk-dashboard-studio); description trimmed to match actual enforcement reality (~37 distinct codes, not the over-promised "50+").
