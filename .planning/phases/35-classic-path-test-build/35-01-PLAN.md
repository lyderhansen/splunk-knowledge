## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| build_flat.js -> viz source | Build script inlines theme.js and wraps source as AMD; untrusted patterns in source could break the bundle |
| validate_viz.sh -> app directory | Validator reads all files in app dir; expects Splunk app structure |
