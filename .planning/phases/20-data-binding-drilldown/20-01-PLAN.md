**viz-blueprints.md — add drilldown notes per viz type (per D-02):**
For each viz type section, add a "Drilldown:" sub-entry after "Expected columns:". The pattern varies by interaction model:

- Single Value Tile (KPI): `Drilldown: simple — entire canvas is one click target. Store primary field in this._clickField. _onClick passes the displayed value regardless of mx/my position.`
- Ring Gauge: `Drilldown: simple — entire canvas is one click target. Store field in this._clickField. _onClick passes the numeric arc value.`
- Status Chip / Badge: `Drilldown: simple — whole chip click. Store statusField in this._clickField. _onClick passes the status string value.`
- Live Ticker: `Drilldown: row-based — each visible row is a click target. Hit-test: use stored row y-positions array. _onClick finds row where my falls in [rowY, rowY+rowH], passes field1 value of that row.`
- Leaderboard: `Drilldown: row-based — each row is a click target. Hit-test: rowIndex = Math.floor((my - headerH) / rowH). Clamp to valid range. Pass name-column value of clicked row.`
- Process Flow / Pipeline: `Drilldown: segment-based — each pipeline stage is a click target. Store stage bounding boxes during render. _onClick iterates stored boxes to find hit; passes stage label value.`
- Donut / Ring: `Drilldown: arc-based — each segment is a click target. Hit-test: Math.atan2(my-cy, mx-cx) to get angle, map angle to segment index. Pass category value of hit segment.`
- Heat Grid / Matrix: `Drilldown: cell-based — each cell is a click target. Hit-test: colIndex = Math.floor((mx - leftPad) / cellW), rowIndex = Math.floor((my - topPad) / cellH). Pass row_label+col_label composite or row_label value.`
- Spark Strip: `Drilldown: simple — entire strip is one click target per series. If single series, whole canvas click passes the series name. If multi-series, store strip row bounds and hit-test rowIndex.`
- Line Chart: `Drilldown: nearest-point — hit-test finds closest data point within threshold pixels (e.g. 20px). Store rendered point positions. Pass value of nearest point or _time string.`
- Radar / Spider Chart: `Drilldown: axis-based — each axis spoke is a click target. Hit-test: Math.atan2 from center to mx/my, map angle to nearest axis. Pass dimension value of hit axis.`
- Needle Gauge (Speedometer): `Drilldown: simple — entire canvas is one click target. _onClick passes the numeric gauge value.`
- Status Matrix / Health Grid: `Drilldown: cell-based — each entity cell is a click target. Hit-test same as Heat Grid. Pass the name field value of clicked entity.`
- Waterfall Chart: `Drilldown: bar-based — each bar is a click target. Store bar x-ranges during render. Hit-test: find bar where mx falls in [barX, barX+barW]. Pass category value.`
- Horizontal Bar List: `Drilldown: row-based — each bar row is a click target. Hit-test: rowIndex = Math.floor((my - topPad) / rowH). Pass label value of clicked row.`
- Data Table (Canvas): `Drilldown: row-based — each data row is a click target. Hit-test: rowIndex = Math.floor((my - headerH) / rowH) + scrollOffset. Pass drilldownField value of clicked row (stored in this._clickField).`
