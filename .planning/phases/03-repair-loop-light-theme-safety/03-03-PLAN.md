                       textDim, textFaint, s1-s5, accent, success, warn, danger, invert
  getTheme('light') → LIGHT object (identical key structure)
  DARK.grid and LIGHT.grid use rgba() strings — all other text/bg/panel/panelHi are #RRGGBB hex.
  DARK and LIGHT are NOT exported directly — only accessible via getTheme().

Pairs to check (per D-11):
  { text: 'text',      bg: 'bg',       threshold: 4.5, level: 'FAIL' }
  { text: 'text',      bg: 'panel',    threshold: 4.5, level: 'FAIL' }
  { text: 'text',      bg: 'panelHi',  threshold: 4.5, level: 'FAIL' }
  { text: 'textDim',   bg: 'bg',       threshold: 4.5, level: 'FAIL' }
  { text: 'textDim',   bg: 'panel',    threshold: 4.5, level: 'FAIL' }
  { text: 'textDim',   bg: 'panelHi',  threshold: 4.5, level: 'FAIL' }
  { text: 'textFaint', bg: 'bg',       threshold: 3.0, level: 'WARN' }   ← informational

Output format (per D-12):
  stdout: '  FAIL CONTRAST: dark.text/bg = 2.10:1 (need 4.5:1) fg=#AAAAAA bg=#FFFFFF\n'
  stderr: FINDING: JSON { type:'FAIL', code:'CONTRAST', file:themeJsPath, message:'...', context:{theme, textKey, bgKey, ratio, threshold} }

Exit codes: 0 = all pass, 1 = FAIL violations (WARN does not cause exit 1), 2 = usage error.

WCAG formula (W3C WCAG 2.1 SC 1.4.3 — exact):
  hexToLinear(c) where c is 0-255: s = c/255; return s <= 0.03928 ? s/12.92 : pow((s+0.055)/1.055, 2.4)
  relativeLuminance(hex): R=parseInt(hex,1,3), G=(3,5), B=(5,7); 0.2126*R + 0.7152*G + 0.0722*B
  contrastRatio(h1,h2): (lighter+0.05)/(darker+0.05) where lighter=max(L1,L2), darker=min

isHex(val): typeof val === 'string' && /^#[0-9A-Fa-f]{6}$/.test(val)
  skip non-hex values silently — do not crash on rgba() tokens.

ES5 CJS discipline: var, function(), no arrow, no const/let, no template literals, shebang line.
Vendor deps: none needed for check_contrast.js (no cheerio, no acorn).

From validate_dash.js structure (use as structural analog):
  #!/usr/bin/env node, var fs = require('fs'), var path = require('path')
  args = process.argv.slice(2); usage guard; file-exists guard; main logic; process.exit.
