- Golden-ratio hero split: 1186px hero / 734px supporting on 1920px canvas
- Shadow card recipe: panel group + 10px inset, fillColor 2-3 stops brighter
- Faux glow recipe: 3-layer (outer 0.04 / mid 0.08 / card 0.6)
- Gutter presets: 20px margins, 16px gaps within groups, 24-32px between sections
- Rhythm zones: sparse (hero KPIs) -> medium (primary viz) -> dense (detail tables)

- Canvas minimum: 1920x1080

From ds-viz-image SKILL.md (verified in RESEARCH.md):
- App-bundled image path: /static/app/{app_id}/images/{filename}.png
- preserveAspectRatio: false for full-bleed backgrounds
- Position at structure[0] for z=0 (behind everything)

From ds-viz-rectangle PATTERNS.md (verified in RESEARCH.md):
- Shadow card: fillColor brighter than canvas, fillOpacity 1.0, rx 8
- Faux glow: 3 nested rects with decreasing opacity
- Position before panels in structure array (earlier = further back)
