# splunk.image — authoritative options reference

Source: `@splunk/visualization-schemas@28.6.0` — extracted from `optionsSchemas/splunk.image.js`.

**3 documented options.**

| Option | Type | Default | Enum / pattern | Description |
|---|---|---|---|---|
| `src` | string | `—` |  | URL of an image file. All browser-supported image formats are acceptable. |
| `preserveAspectRatio` | boolean | `false` |  | Set to true to maintain the aspect ratio of an image when the screen is resized. |
| `title` | string | `—` |  | Specify the title used as alternative text for the image. This text is read by screen readers and displayed when the image cannot be shown. |
