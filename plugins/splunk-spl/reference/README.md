# SPL Command Reference

One file per SPL command. Read on-demand when you need full syntax
detail — don't load all files at once.

## How to use

1. Load `spl-gotchas` skill first (it has the command index).
2. Find the command you need in the categorized index.
3. Read the specific `reference/<command>.md` file for full syntax.

## Source

All content extracted from **Splunk Enterprise Search Reference 8.2.12**
(see help.splunk.com for newer versions). Generated 3/20/2026.

## Template

New command files should follow `_template.md`.
