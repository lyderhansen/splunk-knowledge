The divergence is acceptable since they serve different draw functions, but is worth documenting.
- Recommendation: Add a comment to `generate_previews.py` VIZ_TYPE_KEYWORDS noting "Subset of generate_assets.js keyword table — maps to the 7 Pillow draw functions (Gauge/Bars/Line/Table/Kpi/Ring/Generic)" to explain the intentional delta.
