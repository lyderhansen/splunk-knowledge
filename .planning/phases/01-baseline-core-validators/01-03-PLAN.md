    Record the validator exit code and FAIL codes for Build-OK scoring. A viz is Build-OK=Y if the validator finds no FAIL for that viz's formatter and JS checks (B-series and F-series). Build-OK=N if any FAIL was found. Build-OK=? if tarball-only.

    - Settings-OK: did the formatter settings panel work correctly? (Y/N/?)

    Per D-04, a viz PASSES FISR only if Build-OK=Y AND Render-OK=Y AND Settings-OK=Y. Any N on any dimension = FAIL.

    ```
    ...

    **FISR Score:** X / Y vizs = Z%
    ```
