Script contract:
  - Usage: node check_design.js <formatter.html> <visualization_source.js> <theme.js>
  - Exit 0: no FAIL findings (WARNs do not cause exit 1)
  - Exit 1: one or more FAIL findings
  - Exit 2: usage error (fewer than 3 args)
  - stdout: FAIL/WARN lines with two leading spaces
  - stderr: FINDING: NDJSON for FAIL only (WARNs do NOT write to stderr)

emitFail(code, file, message, context):
  - stdout: '  FAIL ' + code + ': ' + message + '\n'
  - stderr: 'FINDING:' + JSON.stringify({type:'FAIL', code, file, message, context}) + '\n'
  - increments failures counter

emitWarn(code, message):
  - stdout: '  WARN ' + code + ': ' + message + '\n'
  - NO stderr write (verified: check_contrast.js line 131 — only FAIL pairs write FINDING:)

cheerio require pattern:
  var cheerio = require(path.join(__dirname, 'vendor', 'node_modules', 'cheerio'));
  var $ = cheerio.load(html, null, false);  // false = no html/body wrapping
