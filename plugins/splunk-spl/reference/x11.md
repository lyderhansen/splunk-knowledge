# x11 — seasonal decomposition of time series

Source: Splunk Search Reference 8.2.12, page 648.

## Syntax

    | x11 <field> [AS <newfield>]

## Examples

```spl
index=main | timechart span=1d count | x11 count
```

Produces trend, seasonal, and residual components.
